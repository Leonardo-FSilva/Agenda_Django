* Comandos no terminal
    python manage.py startproject # Iniciar o servidor
    python manage.py runserver  # iniciar o servidor
    python manage.py startapp blog  # Para criar apps
    python manage.py collectstatic # para carregar arquivos staticos para o deploy
    django-admin startproject Project .  # o ponto final cria o projeto na raiz
    python manage.py makemigrations  # execultar sempre que alterar o models.py
    python manage.py migrate  # sincronizar as migrações de banco de dados
    python manage.py createsuperuser  # criar perfil para o admin
    python manage.py changepassword USERNAME  # trocar a senha caso esqueça
